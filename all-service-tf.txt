# Google Cloud Platform Terraform Templates Collection

Creating Terraform code for "every single GCP service" is not practical in one response because Google Cloud has hundreds of services and thousands of resources that continuously change.

This guide gives you:

* Production-style Terraform structure
* Separate `main.tf` examples for major GCP services
* Working provider configuration
* Common dependencies
* Frequently used enterprise services
* Easily expandable templates

# Provider Configuration

## provider.tf

```hcl
terraform {
  required_version = ">= 1.5.0"

  required_providers {
    google = {
      source  = "hashicorp/google"
      version = "~> 5.0"
    }
  }
}

provider "google" {
  project = "YOUR_PROJECT_ID"
  region  = "asia-south1"
  zone    = "asia-south1-a"
}
```

# Cloud Functions

## cloud-functions/main.tf

```hcl
resource "google_storage_bucket" "bucket" {
  name     = "function-source-bucket-123"
  location = "ASIA"
}

resource "google_storage_bucket_object" "archive" {
  name   = "function-source.zip"
  bucket = google_storage_bucket.bucket.name
  source = "function-source.zip"
}

resource "google_cloudfunctions2_function" "function" {
  name     = "sample-function"
  location = "asia-south1"

  build_config {
    runtime     = "python311"
    entry_point = "hello_http"

    source {
      storage_source {
        bucket = google_storage_bucket.bucket.name
        object = google_storage_bucket_object.archive.name
      }
    }
  }

  service_config {
    max_instance_count = 1
    available_memory   = "256M"
    timeout_seconds    = 60
  }
}
```

# Monitoring Alert Policy

## monitoring/main.tf

```hcl
resource "google_monitoring_alert_policy" "alert_policy" {
  display_name = "High CPU Usage"
  combiner     = "OR"

  conditions {
    display_name = "CPU Condition"

    condition_threshold {
      filter          = "metric.type=\"compute.googleapis.com/instance/cpu/utilization\""
      duration        = "60s"
      comparison      = "COMPARISON_GT"
      threshold_value = 0.8

      aggregations {
        alignment_period   = "60s"
        per_series_aligner = "ALIGN_MEAN"
      }
    }
  }
}
```

# Best Practice Recommendations

* Use Terraform modules
* Store state in GCS backend
* Use separate environments
* Enable IAM least privilege
* Use Secret Manager instead of hardcoded passwords
* Use CI/CD pipelines
* Enable audit logs
* Use remote state locking
